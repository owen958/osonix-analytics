Spotify for Artists Direct API (partial) Live Needs Auth Connect Spotify OAuth above Followers & Popularity = live. Streams = not in API. Monthly listeners = S4A browser only (~24h delay).
Apple Music for Artists API (label-tier only) / Browser Daily (multi/day) Limited 250k+ plays/month required for API. Manual/CSV otherwise. Music Analytics API needs iTunes Connect key. Most indie artists: browser + CSV export.
SoundCloud Direct API (public stats) Near Real-Time Needs Client ID Register app at developers.soundcloud.com → get Client ID Public follower + track count via API. Detailed play analytics = browser Pro only.
Chartmetric Browser only / Manual Daily Manual Copy metrics from chartmetric.com into the manual entry fields, or drag in a CSV export No API access on current plan. Manual entry takes ~30 seconds. Best used as a daily scorecard check.
